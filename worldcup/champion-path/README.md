# 2026 美加墨世界杯 · 我的夺冠之路

一个自包含的单文件交互网页。双击 `index.html` 即可离线打开，无需任何构建、无任何外部 CDN/网络资源（emoji 为字符，背景图为同目录 `stadium-bg.jpg`）。

## 玩法
在 FIFA 官方风格的镜像对阵图上逐场点选晋级球队：左半区 8 场 R32 在最左、向右收拢，右半区 8 场在最右、向左收拢，正中央是冠军大格（下方含三四名决赛）。选满决赛后中央以举杯视觉 + 礼花展示你选出的世界杯冠军。

## 功能
- 单张镜像对阵树（左右半区向中心收拢），CSS Grid 9 列布局 + SVG 肘形连接线把每两场连到下一轮；整体自适应缩放，过窄时缩到下限后横向滚动，保证树形不塌。
- 球场主题海报背景（同目录 `stadium-bg.jpg`）+ 深色遮罩 + 半透明玻璃卡片 + 金色描边。
- 点击晋级有金色描边 + 缩放动画，落败方淡出，胜者自动填入下一轮对应槽位。
- 改选上游时，自动级联清空所有依赖该结果的下游选择，保持逻辑自洽。
- 顶部进度条 / 计数（已选 N/31 场）。
- 保存到 localStorage（key：`wc2026_champion_path`），刷新自动恢复。
- 清空重选。
- 导出：原生 Canvas 绘制一张分享卡片 PNG（标题 + 冠军 + 四强 + 决赛对阵 + 进度），点击下载。
- 复制夺冠之路文本（兜底）+ 复制分享链接（选择序列化到 URL hash，他人打开即还原）。

## 说明
- 对阵与晋级树为「修正版」镜像拓扑：德国/法国同左 QF 区、西班牙/葡萄牙同左 R16 区、巴西与阿根廷均在右半区；决赛 = 左半区冠军 vs 右半区冠军。
- 数据仅保存在本机浏览器，不上传任何服务器。
